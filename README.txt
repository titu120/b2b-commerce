Free WooCommerce B2B & Wholesale Plugin with user management, pricing, and product control. Upgrade to B2B Commerce for advanced features.

== Description ==

B2B Commerce is a powerful free WordPress plugin that transforms your WooCommerce store into a comprehensive B2B and wholesale platform. Perfect for businesses that need to manage different customer types, pricing tiers, and user approvals.

= Key Features (Free Version) =

* **B2B User Management** - Create and manage B2B customer accounts with approval workflows
* **Role-based Pricing** - Set different pricing for wholesale customers, distributors, and retailers
* **User Approval System** - Control who can access your B2B features
* **Order Management** - Track and manage B2B orders
* **Catalog Mode** - Hide prices and disable purchasing for non-B2B customers
* **Email Templates** - Customize email notifications for B2B customers
* **Analytics & Reporting** - Track B2B performance and user activity
* **Import/Export** - Bulk import/export users and pricing rules
* **REST API** - Integrate with external systems

= Pro Features (B2B Commerce) =

* **Quote Management System** - Receive and manage quote requests
* **Product Inquiry System** - Handle customer product inquiries
* **Bulk Pricing Calculator** - Interactive calculator for bulk pricing
* **Advanced Email Notifications** - Enhanced email system
* **Priority Support** - Dedicated support for Pro users
* **Regular Updates** - Continuous feature updates
* **Custom Integrations** - Advanced integration options

= Perfect For =

* Wholesale businesses
* B2B distributors

* Retailers with B2B operations

= Easy Setup =

1. Install and activate the plugin
2. Configure B2B settings in the admin panel

4. Start managing your B2B customers

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/b2b-commerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the B2B Commerce settings page to configure the plugin
4. Create B2B user roles and set up pricing rules
